README

TechCrunch MySpace OAuth tester app.

h2. Requirements

* pear http_request library "pear.php.net":http://pear.php.net/
	* this is used for making http requests
* php 5.2+ _or_ "json-php":http://mike.teczno.com/json.html
	* used for json parsing

h2. Installation

# Install pear http_request library: sudo pear install http_request
